## react intro

    What is React?

    1) React is a JavaScript library (not a framework) developed by Facebook (Meta) for building user interfaces (UIs), especially Single Page Applications (SPAs).

    2) It helps developers build reusable UI components that update efficiently when data changes.

    Instead of reloading a whole webpage when something changes, React updates only the parts of the UI that need to change.

    -------
    Why do we need React?

	It makes UI development faster, cleaner, and more efficient.

	Provides a better user experience with smooth updates without page reloads.

## usestate()

     usestate is a react hook that allows you to add state (data that can change over time)
     your a functional component.

     Normally in js  functions, variables disappear after the function runs, but with usestate,
     reat remembers values b/w re-renders.

        syntax:

            const[state,setState]=usestate(initiateValue)

            const[count,setCount]=usestate(222)

            state-the current value of the state variable.
            setState-- a function that update the  state.
            initiateValue--- the starting value of the state.

## useEffect()

    In react ,useEffect is a Hook that lets you perform side effect in 
    functional component.
    Side effects are task that happends outside  the normal react rendering process,
    Like:       
            Fetching data from an API
            Updating the browser DOM manualluy
            starting/stoping timers.

    syntax:

            useEffect(()=>{
                //code
            },[count ,dependency 2...])


        [] ---> if the dependency is empty means ,it runs only once at initially.

## useRef()

    useRef is a react hook that gives you  a box (.current) you can store anything
    in it -- number,string,object,function,even  DOM elements.

    That value persists between renders.

    Updating it does not causes your component to re-render.

    {current:value}

    syntax:

        const variablename=useRef(initiateValue)

## useReducer()

    syntax:

        const [state,dispatch]=useReducer(reducerFunction,initialState)

        state-- current state (like usestate value).
        dispatch(action)---a function to tell reducer what to do.
        reducerFunction(state,action)--- the logic that returns a new state.
        initialState-- starting value.
